# Algorithm
## Dynamic Programming (동적계획법)

- Bottom-Up 방식
반복문 이용
제일 작은 값부터 구해나감

- Top-Down 방식
재귀함수 이용
하나의 문제를 여러 하위 문제로 쪼갬
하위 문제의 결과를 저장하여 상위문제에서 사용
