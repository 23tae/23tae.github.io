# Problem
<https://www.acmicpc.net/problem/11021>

# Solution
주어진 횟수를 통해 for문을 구성하였다.
print문의 양식이 복잡하기 때문에 문자열 포맷팅을 활용하여 표현하였다.
# code
```py
a = int(input())

for i in range(a):
    b = list(map(int, input().split()))
    print("Case #%d: %d" %(i+1,b[0]+b[1]))
 ```
