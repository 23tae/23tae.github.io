# Problem
<https://www.acmicpc.net/problem/2884>

# Solution
```python
h,m = map(int, input().split())
if m>=45:
    print(h,m-45)
else:
    if h!=0:
        print(h-1,m+15)
    else:
        print(23,m+15)
```
# Memo
조건문을 활용해 H와 M이 범위 밖으로 나가는 것을 방지한다.
