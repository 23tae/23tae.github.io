# Solution
## code
```python
x = int(input())

if 90<=x<=100:
    print("A")

elif 80<=x<=89:
    print("B")

elif 70<=x<=79:
    print("C")

elif 60<=x<=69:
    print("D")

else :
    print("F")
```
