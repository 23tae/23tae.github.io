# Problem
<https://www.acmicpc.net/problem/10952>

# Solution
while문을 사용해서 계속 입력받음.
문제의 조건인 입력값 0 0 일 때 프로그램을 종료시키는 부분은 if문에 적용함.

## code
```py
while True:
    a = list(map(int,input().split()))
    if a[0]==a[1]==0:
        break
    else:
        print(a[0]+a[1])
```
