# Problem
<https://www.acmicpc.net/problem/1094>

# Solution
```py
x = bin(int(input()))
print(x.count("1"))
```

# Memo
문제에서 설명하는 과정대로 풀어내려고 했더니 너무 복잡했다. 따라서 과정을 단순화 했더니 2진수에서의 1의 개수를 구하는 것으로 정리됐다.
우선 bin 함수를 사용하여 입력된 10진수를 2진수로 변환한 후 count 를 통해 1의 개수를 알아냈다.
