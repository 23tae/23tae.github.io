By Taehoon
1 min read

Ubuntu에서의 해결방법은 다음을 참고.

① .zshrc 파일 수정

홈 디렉터리에서 .zshrc파일을 수정하여 하단에 다음과 같은 라인을 추가한다.

1

code () { VSCODE_CWD="$PWD" open -n -b "com.microsoft.VSCode" --args $* ;}
