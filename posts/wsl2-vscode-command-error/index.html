WSL2에서 vscode가 실행되지 않는 문제 (code 명령어)

Posted Feb 5, 2022 Updated Dec 4, 2025
By Taehoon Kim
1 min read

WSL2에서 Ubuntu를 실행하여 code . 명령어를 입력했을 때 not found가 출력되는 문제에 관한 해결방법을 담고있음.

Issue

평소처럼 terminal을 실행시키고 홈 디렉토리에서 code . 명령어를 쳤는데 /mnt/c/Users/~/bin/code: not found 라는 문구가 출력됐다. 윈도우에서 직접 vscode를 실행시키니 프로그램의 문제는 아니었다. 한참을 이 문제로 고생하다 구글링을 해보니 환경변수 설정이 해제되어 생긴 문제로 보였다.

Solution

홈디렉토리에서 .profile을 편집기로 실행시킨 후 맨 아래에 PATH="$/mnt/c/Users/(유저이름)/AppData/Local/Programs/Microsoft\ VS\ Code/bin" 다음과 같은 환경변수를 추가하면 해결된다.
